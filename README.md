# Tolga Ersan - Industrial Design Portfolio

This is a personal industrial design portfolio website built for Tolga Ersan. It is designed and developed by Muhammet Özmen using Jekyll.

## Live Site
The portfolio is live and can be visited at: http://www.tolgaersan.com

## Features and Tech Stack
* Static Site Generator: Jekyll
* Target Audience: Industrial design showcases, product renders, sketching portfolios, and engineering projects.
* Design Philosophy: Minimalist, clean, and highly visual to place the design work at the absolute center of attention.
